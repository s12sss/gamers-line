# Gamer's Line — X運用 引継ぎ資料（通常投稿 + 広告）

## サイト概要

**サイト名**: Gamer's Line（ゲーマーズライン）
**URL**: https://gamers-line.jp/
**ジャンル**: ゲーマー向け光回線 診断・比較メディア
**ローンチ**: 2026年5月4日

### コンセプト
「オンラインゲームをプレイするユーザーが、自分の環境に最適な光回線を見つけるメディア」。Ping値・安定性・料金の3軸でゲーマー目線の評価を行う。他の光回線比較サイトとの最大の違いは、キャッシュバック額でなくPing値と安定性を軸にしていること。

---

## ターゲット

- オンラインゲームをプレイするユーザー全般（ジャンル不問）
- 道具・環境にこだわるタイプのゲーマー
- 「ラグが気になる」「回線を見直したい」と思っているユーザー
- 年齢層：10代後半〜30代
- プラットフォーム：PC・PS5・Switch・スマホ問わず

**注意**: FPS専用サイトではない。MMO・格ゲー・マルチプレイ・カジュアルゲーマーも対象。

---

## 主要機能・コンテンツ

| 機能 | URL | 説明 |
|------|-----|------|
| 回線診断ツール | /diagnosis | ゲームジャンル・住居・地域などを入力して最適回線を提案（30秒） |

| コラム | /column | ゲーマー向け回線・ネットワーク解説記事 |

## 通常投稿 運用方針

### アカウントの方向性
- 名前：Gamer's Line（またはGamer's Line | ゲーマー向け回線診断）
- キャラクター：ゲームに詳しい中立的な情報発信者。売り込み感を出さず、「知ってると得する情報を届ける」スタンス
- 口調：です・ます調ベース。固すぎず、ゲーマーと自然に話せる温度感

### 投稿頻度
- 最低週3〜4本。毎日投稿できればベター
- 広告クリエイティブ的な投稿は全体の2〜3割まで。残りは純粋に有益な情報

### コンテンツの種類（投稿ネタの柱）

| 種類 | 例 | 割合 |
|------|----|------|
| 知識・豆知識 | 「Ping値とジッターの違い」「なぜWi-Fiはゲームに向かないか」 | 40% |
| 比較・データ | 「各回線のPing値比較（表付き）」「独自回線とコラボ光の違い」 | 25% |
| 診断誘導 | 「回線に悩んでる人は30秒診断してみて→」 | 15% |
| 時事ネタ連動 | 新作ゲームリリース時に「〇〇をプレイするなら回線はこれ」 | 15% |
| エンゲージメント | 「今使ってる回線は？」アンケートなど | 5% |

### 投稿のトーン・NGライン

**やること**:
- 数値を使った具体的な訴求（「Ping12msのNURO光」など）
- ゲーマー目線の言葉（ラグ・ドロップ・安定性）
- 短く読みやすいフォーマット（箇条書き・改行多め）

**やらないこと**:
- キャッシュバック金額の具体的な数字（月次で変わるため）
- 「最強」「絶対」「No.1」などの断定表現
- FPS専用・FPSゲーマー限定という絞り込み（全ゲーマーが対象）
- 頻繁な申し込み誘導（売り込み感が強くなる）

### ハッシュタグ
投稿の内容に応じて2〜4個程度。多用しない。
- `#光回線` `#ゲーミング回線` `#Ping` `#オンラインゲーム`
- ゲームタイトル関連：`#VALORANT` `#ApexLegends` `#FF14` など内容に合わせて

### 投稿例
```
Wi-Fiと有線の違い、ゲームだとこんなに出る。

Wi-Fi（5GHz）: Ping 8〜25ms、ジッター大
有線LAN（Cat6）: Ping 5〜15ms、ジッター小

ゲーム中のラグの大半はルーター〜PC間の問題。
回線を変える前にまずケーブルを確認してみて。

詳しくはこちら→ [記事URL]
```

## X広告 運用方針

### 目的
1. サイトへの認知流入（診断ツール利用促進）
2. アフィリエイト経由の回線申し込み誘導

### ターゲティング方針
- キーワードターゲティング：「ラグ」「光回線」「Ping」「回線 重い」「オンラインゲーム 遅延」
- フォロワーターゲティング：ゲーム系アカウントのフォロワー（VALORANTJapan, ApexLegendsなどの公式アカウント周辺）
- 興味関心：オンラインゲーム・FPS・格ゲー・eスポーツ

### 予算感
- 初期テスト：¥10,000程度
- Amazonギフト券プレゼントキャンペーンと同時実施（7月以降）

### 訴求の方向性

**やること**:
- 「ラグが気になるなら回線を診断してみよう」的な問題提起型
- 診断ツールへの誘導（無料・30秒という訴求）
- 「自分に合った回線がわかる」という利便性訴求
- Ping値など具体的な数値での訴求

**やらないこと**:
- 特定のキャッシュバック金額の明記（更新が必要になるため）
- 「FPS専用」「FPSゲーマーに」という絞り込み（ターゲットが狭くなる）
- 誇大表現・保証的な言い回し

### クリエイティブの方向性
- トーン：クール・ゲーミング感・信頼感（ガチャガチャしない）
- デザイン参考：サイトのサイバーパンク調ダークデザイン（#050508背景・シアン#00E5FFアクセント）
- コピー例：
  - 「そのラグ、回線のせいかも。30秒で診断→」
  - 「ゲームの回線、ちゃんと選んでる？Ping値・安定性で比較できるサイト」
  - 「光回線を選ぶならPing値で選ぼう。無料診断はこちら→」

### KPI
- CTR（クリック率）：1.5%以上を目標
- 診断完了率：LP流入の30%以上
- CPA（1申込みあたりコスト）：¥3,000以下

## 注意事項

- 光回線のリスティング広告（Google/Yahoo）はASP規約で禁止のためNG
- X広告はASP規約の対象外のため実施可能
- キャッシュバック額は月次で変動するため、広告クリエイティブに具体的な金額を入れないこと
- 「最安」「業界No.1」等の優位性表現は薬機法・景表法観点から避けること
