## 安定性スコアの見方

サイト独自の100点満点指標。Ping変動幅・パケットロス率・夜間帯混雑による速度低下率を総合評価。

| スコア | 評価 |
|--------|------|
| 90〜100 | 非常に安定。夜間・休日でもPingが崩れにくい |
| 80〜89 | 安定。一般的なゲームプレイには十分 |
| 70〜79 | 普通。混雑時間帯にPingが上昇することがある |
| 〜69 | やや不安定。FPS等の競技プレイには注意 |

## 注意事項（記事に書くとき）

- Ping値・速度は**推計値（期待値）**。実環境により異なる
- 料金は**税込・標準月額**。キャンペーン適用で変動する
- マンション料金は**光配線方式**の場合。VDSLはさらに制限あり
- NURO光のカバーエリア: **関東・東海・関西・中国・九州**（北海道・東北・四国・沖縄は非対応）
- 地域系回線（コミュファ・eo・ピカラ・BBIQ・メガエッグ）は**特定エリアのみ**
