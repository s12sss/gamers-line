# SNS投稿戦略

Gamer's Line とギアセレクトを育てるためのSNS運用メモ。  
ClaudeにX投稿案を作らせる時は、この方針を前提にする。

## 2026年9月までの優先方針

9月までは Gamer's Line の回線サイトとしての認知と信頼を育てる期間にする。

投稿比率:

- 回線ネタ: 8割
- note / ギアセレクト宣伝: 2割

PC周辺機器やデスク周りの話題は出してよいが、9月までは「ゲーム環境づくりの一部」として扱い、回線アカウントの軸を崩さない。

## 1日の投稿数

基本は1日2投稿。

### 投稿1: URLなしの豆知識投稿

目的:

- フォロー理由を作る
- タイムラインで読まれやすくする
- 売り込み感を出さずに信頼を積む

内容:

- Ping改善
- ラグの原因
- ルーター選び
- マンション回線の注意点
- 10G回線の向き不向き
- 回線速度とゲームの関係

URLは基本入れない。  
最後に無理に誘導しない。保存・共感・返信を狙う。

### 投稿2: URL付きの導線投稿

目的:

- Gamer's Lineの記事に送る
- 無料診断に送る
- キャンペーン記事や比較記事に送る

内容:

- 診断ページ
- 回線比較ページ
- キャンペーンページ
- NURO光 / GameWith光 / auひかりなどの記事
- エリアSEO記事

URLは1つだけ。  
1投稿で複数リンクを貼らない。

## 投稿比率の目安

1週間14投稿の場合:

- 回線豆知識: 6投稿
- URL付き回線導線: 5投稿
- note / ギアセレクト: 3投稿

note投稿は週2-3回まで。  
楽天ROOMや買ってよかったものは、Xの主役にしすぎない。

## Xアカウントの立ち位置

Xは「ゲーム環境づくりの入口」。  
ただし9月までは回線を主役にする。

主な読者:

- FPSでラグに悩んでいる人
- 回線選びで失敗したくない人
- NURO光、GameWith光、auひかりなどで迷っている人
- Ping値を下げたい人
- ルーターや10G回線が必要か迷っている人
- ゲーム部屋やPC環境も少しずつ整えたい人

## アカウント人格

ラグに悩んで実家を出たコアゲーマー。

高級機材を見せびらかす人ではなく、ゲーム環境をちゃんと整えたい人。回線、Ping、ルーター、PC周辺機器、デスク周りを実体験ベースで語る。

### キャラの軸

- 本音で言う
- 無理に高いものをすすめない
- 初心者にも分かる言葉にする
- 失敗談も出す
- 「環境づくり」を一緒に進める感じにする
- 回線の話を難しくしすぎない

### 避けること

- 露骨なアフィ誘導
- 煽りすぎる断定
- 何でもおすすめする投稿
- AIっぽい整いすぎた文章
- 回線、PC、noteを1投稿に詰め込みすぎる
- 毎投稿で無料診断に誘導する

## Xプロフィール案

ラグに悩んで実家を出たコアゲーマーが、ゲーム環境づくりを発信。  
回線選び・Ping改善・ルーター・PC周辺機器・デスク周りを本音レビュー。  
最適な回線が分かる『無料診断』はこちら👇

ギアセレクト

プロフィール:

いい感じのゲーム部屋を作りたくて、いろいろ試しています。  
デスク周り、ガジェット、セールで買ってよかったものなど。失敗したものも普通に書きます。  
ゲーマー向け回線サイト「Gamer's Line」も運営中。

## 投稿カテゴリ

### 1. 回線豆知識

URLなし。9月までの主力。

狙い:

- フォローしてもらう
- 回線に詳しい人として認知される
- 返信や保存を狙う

テーマ例:

- Pingと速度の違い
- ラグの原因
- ルーターの置き場所
- Wi-Fiと有線の違い
- IPv6/IPoE
- マンション回線の混雑
- 10G回線の必要性
- 夜だけ重い原因
- FPSで見るべき回線指標
- 回線を変える前に見るべきこと

### 2. 回線記事へのURL付き投稿

Gamer's Lineの記事へ送る投稿。

狙い:

- 記事アクセス
- 無料診断
- アフィリエイトCV

対象ページ:

- 無料診断
- 回線比較
- キャンペーン
- 各回線レビュー
- エリア記事
- 速度チェック

ルール:

- URLは1つだけ
- 冒頭で悩みを言う
- 中盤で判断基準を出す
- 最後に「詳しくまとめた」でURL
- 強引に売らない

### 3. note / ギアセレクト

9月までは2割に抑える。  
Amazonアフィ、楽天セール、楽天ROOMへの導線。ただし表では「買い物ログ」「ゲーム部屋づくり」として見せる。

投稿対象:

- 楽天セール: 月1回
- 買ってよかったもの: 月1回
- ギア特集: 不定期
- 個別商品レビュー: 買った時
- 楽天ROOM: Xプロフィールか固定導線

Amazonは投稿文で強く出しすぎない。  
「Amazonで買えます」より「ゲーム部屋に置いてよかった」の見せ方を優先する。

### Claudeが投稿作成前に確認すること

投稿案を作る前に、以下がそろっているか確認する。

- 宣伝したいURL
- ユーザーアクションの目的
- 短文か長文か

どれか1つでも抜けている場合、Claudeは投稿文を作らず、先に質問する。

質問例:

```text
投稿を作る前に3点だけ確認したいです。

2. ユーザーに取ってほしい行動は何ですか？ 例: 無料診断、記事閲覧、比較、note購読、楽天ROOMクリック
```

確認例:

```text

1. テーマは回線速度、Ping、ルーター、10G、マンション回線のどれ寄りですか？
```

## AIっぽさを消すための書き方

投稿文は、きれいにまとめすぎない。  
少し話し言葉寄りで、管理人がタイムラインに普通に書いている感じにする。

### 基本ルール

- 1文を短くする
- きれいに締めようとしない
- 少し本音っぽく言う
- 強く断定しすぎない
- 読者に説教しない
- 毎回同じ型で終わらない
- 「売る」より「気づきを置く」感じにする

### 使ってよい崩し方

- 文頭に「正直」「これ」「地味に」「個人的に」を使う
- 「人による」「まずここ」「これは普通にある」を使う
- 「〜なんですよね」「〜でいいと思う」も使う
- 体験談から入る
- あえて余白を残して終える
- 口語を少し混ぜる

### 避ける言い回し

- 「〜することが重要です」
- 「〜をおすすめします」
- 「ぜひ参考にしてください」
- 「この記事では〜を解説します」
- 「結論から言うと」
- 「以下のポイントを押さえましょう」
- 「快適なゲーム環境を実現しましょう」
- 「あなたに最適な〜」
- 「今すぐチェック」
- 「知らないと損」
- 「完全攻略」

### Claudeへの追加指示

投稿作成時は、必要に応じて以下を追加する。

```text
文章は少しラフでOK。
きれいに締めようとしない。
「重要です」「おすすめします」「ぜひ参考にしてください」は使わない。
タイムラインに普通に流れてきても違和感ない文にして。
売り込みより、気づきや本音を優先して。
```

## URLなし豆知識の構文

### 速度よりPing型

回線速度が速いのにゲームでラグい人、けっこういる。  
ゲームで大事なのは「何Mbps出るか」より、Pingと安定性。  
特に夜だけ重いなら、速度より混雑や経路を疑った方がいい。

### 失敗談型

昔は「速い回線にすれば全部解決する」と思ってた。  
でもゲームだと、速度よりPingと安定性の方が体感に出る。  
回線選びで見る場所を間違えると、乗り換えてもラグが残る。

### ルーター型

回線を変える前に、ルーターの置き場所も見た方がいい。  
床置き、棚の奥、電子レンジの近くは普通に不利。  
Wi-Fiでゲームするなら、回線より先にここで損してることがある。

### 有線推奨型

FPSをやるなら、まず有線にできるかを考えた方がいい。  
Wi-Fiが悪いというより、無線は環境の影響を受けやすい。  
Pingを安定させたいなら、派手な機材よりLANケーブルの方が効くこともある。

### 10G慎重型

10G回線は強いけど、全員に必要ではない。  
FPSだけなら1Gでも十分なことは多い。  
配信、家族利用、大容量DLが多い人は10Gの恩恵が出やすい。

## URL付き導線投稿の構文

### 無料診断導線

回線選びは「一番速い回線」を探すより、用途に合う回線を選ぶ方が大事。  
FPS、配信、家族利用、マンションか戸建てかで向き不向きが変わる。  
迷う人向けに、ゲーマー用の無料診断を作りました。  
URL

### 比較記事導線

NURO光とGameWith光、どっちがいいかは人による。  
速度だけで見ると迷いやすいけど、Ping、エリア、開通までの待ち時間、料金で見ると選びやすい。  
ゲーマー目線で比較しました。  
URL

### キャンペーン導線

回線を乗り換えるなら、キャンペーンは見た方がいい。  
ただ、キャッシュバック額だけで選ぶと失敗することもある。  
ゲーム用途で見るべきポイントと一緒にまとめています。  
URL

### エリア記事導線

同じ回線でも、住んでいる地域で選び方は変わる。  
エリア内か、工事できるか、マンションで使えるか。  
地域別にゲーマー向け回線を整理しました。  
URL

## バナー運用

ギアセレクトのバナーは用途別に使う。

- 楽天セール: 月1回。`RAKUTEN SALE`
- 買ってよかったもの: 月1回。`BEST BUY`
- ギア特集: 不定期。`GEAR FEATURE`
- 個別商品レビュー: 商品購入時。`DEVICE REVIEW`
- 自由テキスト: 告知や汎用サムネ

Amazonはバナー上で強く出さない。  
表では「ギア特集」「買ってよかったもの」として見せ、本文内のリンク先として使う。
